# neon-canvas
A playground for experimental UI components and fluid animations. Built for the modern web.

## what is this?
honestly, just a messy laboratory where i can break things, test weird ideas, and build UI components that look cool. standard web design can get pretty boring, so i'm using this space to experiment with liquid-smooth animations, heavy neon glows, and interfaces that feel alive when you interact with them.

## things i want to build here:
- micro-interactions that actually feel satisfying to click
- experiments with frosted glass and heavy dark-mode aesthetics
- some hardware-accelerated canvas experiments if i get ambitious

## tech stack (for now):
- vanilla html/css/js
- probably gsap or framer motion once things get complex
- whatever random npm package catches my eye
